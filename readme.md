simple YouTube MP3/MP4 downloader built with yt-dlp, designed to work as a Raycast script.

# Installation
## Requirements
- ffmpeg executable in project directory

---
### Using Python

- Install Python 3.13
- Install yt-dlp package using pip:
    - ```pip install yt_dlp```

- Add the script to Raycast Scripts directory and run it from Raycast 😺

### 🛠️ Usage (Raycast)
1. Run command `kyd`
2. Provide
    - URL -> Youtube URL
    - Format -> `mp3` or `mp4`

Example:
`kyd https://www.youtube.com/watch?v=dQw4w9WgXcQ mp3`

## Output directory
- Files are saved to: ` ~/Videos/kyd_output`
